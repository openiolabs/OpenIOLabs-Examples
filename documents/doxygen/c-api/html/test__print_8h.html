Detailed Description

ScriptML lightweight printing operations.

These functions can be used to print numbers and strings using ScriptML's builtin printing capabilities, removing the need to include <stdio.h> and hence process the printf algoithm. This is used in test scripts to speed up printing. For reference builds, a stdio-based alternative is swapped in.
