#define set_exception_handler (   TYPE,
  HANDLER 
)    ___SetExceptionHandler( exception_type:TYPE, handler:*(HANDLER) )

Set a handler for script exceptions (used as a substitute for exception handling in C)

Pass in the exception type (as a type) and a pointer to a handler function.

Handler will be called as soon as the exception occurs (i.e. at the point of throwing). If it requrns, script execution will continue from the failed operation. Return values and other putputs from the failing operation will be undefined.

Typically, the script will set an error flag and/or an LWM2M error resource, OR just halt or restart the script.
